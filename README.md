A command-line inventory management system that allows users to track products and their quantities in a store.
Built with Python using dictionaries and functions.


Features:

Add new products with quantities
Remove products from inventory
Update product quantities
Check stock levels for specific products
Display complete inventory


Error Handling:

Product not found: When trying to remove or update a product that doesn't exist, the system displays "Product not in inventory."
Invalid commands: Unrecognized commands display "I don't recognize that command."
Quantity validation: Quantities must be entered as numbers (integers)

All quantities are cumulative when using updateq
Product names are case-sensitive
Inventory resets when program is restarted (data not saved) ! ! ! !
